# A Cars Sales Report using powerbi
>[!NOTE]
>Dataset is taken from kaggle.
>PowerBi is pulling data from google sheets, make sure to refresh the data on the first time you open.

>[!NOTE]
>This personal project is meant to be used for portfolio and showcasing purposes.

## Utilized Systems
1. ETL Process.
2. Different types of charts which interact with each other.
3. Page Slicers for specific insights.
4. Drill through pages for a deeper dive.
   
## Analysis Questions
1. Which Dealers sold the most cars?
   - Which models are popular?
   - Which regions are the hotspots for car purchases?
   - What is the average preferred price range of the customer purchases relative to their annual income?
2. Which Company sold the most units?
   - What is the most popular model sold?
   - What is the most popular body style of the most popular model sold?
   - What is the preferred color of the model?
   - What is the preferred transmission type of the model?
   - What is the preferred type of each gender?
